# WishMS music player
WishMS is a basic design of a simple music player it includes a list of songs that I've personally placed along with the cover. This project is made totally for practicing purpose. WishMS includes a list of 9 songs which are my personal favourite user's can also make this project or try it by forking or cloning this repository and adding their own songs and covers.

## Features 
- Dynamic play buttons 
- Progression bar with seek function 
- List of nine songs with cover
- songs include their own play button
- Display of song title, artist, and album artwork

## How to get started
- Clone the repository:
 https://github.com/Nottoishenshu/WishMS

- Navigate to the project folder:
cd WishMS

-Open index.html in your browser.
No additional installation or dependencies are required.

## Challenges I faced 
While making this project the first problem I faced was with the progression bar the size seemed way smaller than required so i had a problem there, after that there seemed to be a problem with the font awesome the fonts get turned into a SVG directly so inorder to prevent that I deleted the font awesome and used regular buttons but users can also include a font awesome as their play button from **(https://fontawesome.com/)**. After this the cover and the file not loading the file was misplaced so when users want to use this they need to be careful with the file placement and name and on which folder they have included the mp3 files and jpg files. Futher more I got to learn a lot from this project and here in the future would like to include more features.
